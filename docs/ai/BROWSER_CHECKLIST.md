# BROWSER_CHECKLIST.md

京都サンガF.C.日程表HTMLを、人間が実ブラウザで毎回すばやく確認するための軽量チェックリストです。

## 目的

静的検証では見つけにくい、実ブラウザ上の大きな表示崩れ、主要操作の不具合、赤いJavaScriptエラーだけを短時間で確認します。

毎回の確認は「全部を細かく見る」前提にせず、5分以内のスモークチェックとして実施します。小さな見た目の差分や詳細なアクセシビリティ確認は、UIを大きく変更した時だけ別途行います。

## 毎回の確認は5分以内にすること

- 1回の確認で完璧を目指さず、公開前・PR確認時に毎回続けられる軽さを優先します。
- 問題がなければ、各項目は「問題なし」とだけ記録して構いません。
- 問題があった場合だけ、画面幅、ブラウザ、再現手順、期待結果、実際の結果を簡潔に残します。
- 詳細確認が必要だと判断した場合は、通常確認とは分けて後続タスクにします。

## 通常確認: 7項目

- [ ] スマホ幅で開いて、主要表示が大きく崩れていない
- [ ] PC幅で開いて、主要表示が大きく崩れていない
- [ ] 縦幅が大きい試合カードの行で、隣接カードの背景・縦帯・日付欄が不自然に短く見えない
- [ ] 表示列変更が動く
- [ ] 使い方ダイアログが開閉できる
- [ ] LocalStorage削除ボタンが動く
- [ ] ブラウザコンソールに赤いJavaScriptエラーが出ていない

## 詳細確認はUIを大きく変更した時だけ行う

次のような変更をした時だけ、通常確認に加えて詳細確認を行います。

- HTML構造、CSSレイアウト、JavaScriptの操作仕様を大きく変更した時
- LocalStorageの保存・復元・削除に関わる処理を変更した時
- ダイアログ、ボタン、表示列変更など主要操作を変更した時
- 日程データの生成方法やJSON表示ロジックを変更した時
- アクセシビリティに影響しそうな変更をした時

### 詳細確認が必要な場合の参考項目

通常確認で不足する場合だけ、以下を必要な範囲で確認します。

- 表示列設定が保存され、再読み込み後も復元される
- キーボードで主要ボタンに移動でき、フォーカス表示が見える
- 使い方ダイアログをキーボードで開閉できる
- 免責事項、使い方、LocalStorage説明文が意図せず欠けていない
- 日程数字、対戦相手、会場、時刻の表示が意図どおりである
- フッターや注記が大きく崩れていない
- スマホ幅・PC幅以外の中間幅でも大きな破綻がない
- ネットワーク読み込みエラーがある場合、表示への影響を説明できる
