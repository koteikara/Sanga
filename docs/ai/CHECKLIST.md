# CHECKLIST.md

## データ確認

- [ ] 試合日程データを意図せず変更していない
- [ ] 節番号を意図せず変更していない
- [ ] 対戦相手・会場・時刻を意図せず変更していない

## 表示確認

- [ ] 実ブラウザでの目視・操作確認は `docs/ai/BROWSER_CHECKLIST.md` の5分スモークチェックを参照する
- [ ] PC幅で主要表示が大きく崩れていない
- [ ] スマートフォン幅で主要表示が大きく崩れていない
- [ ] 縦幅が大きい試合カードでも同じ行の表示が大きく崩れていない
- [ ] 日程数字が詰まりすぎていない
- [ ] フッター表示が崩れていない
- [ ] 免責事項が表示されている

## 機能確認

- [ ] 表示列変更が動く
- [ ] 使い方ボタンがボタンらしく見える
- [ ] 使い方ダイアログが開く
- [ ] 使い方ダイアログが閉じる
- [ ] LocalStorage削除ボタンが動く
- [ ] LocalStorage説明文が適切に残っている

## アクセシビリティ確認

- [ ] キーボードで主要ボタンに移動できる
- [ ] フォーカス表示が見える
- [ ] ボタン名が理解できる
- [ ] 色だけに依存した説明になっていない
- [ ] ダイアログの開閉が理解しやすい

## JavaScript確認

- [ ] JavaScriptを変更する場合は、`docs/ai/JS_CHANGE_CHECKLIST.md` も参照する。
- [ ] ブラウザコンソールに明らかなエラーがない
- [ ] 既存LocalStorageキーを変更していない
- [ ] 保存済み設定が意図せず消えない
